# Политика безопасности MOZG Universe

## Конфиденциальная информация
Никогда не коммитьте:
- Ключи API (.env файлы)
- Пароли и учетные данные
- Сертификаты и ключи шифрования
- Личные данные пользователей

## Что делать при утечке
Если вы случайно закоммитили секреты:
1. Немедленно сгенерируйте новые ключи
2. Удалите файл из истории Git
3. Уведомите команду security@mozg-universe.com
