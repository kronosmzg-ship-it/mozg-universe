
## Структура проекта
- `/documentation` — все исследования, поставщики, стандарты, бизнес-план
- `/brand-assets` — логотипы, брендбук, шрифты
- `/ai-prompts` — коллекция промптов для генерации изображений, текстов, видео
- `/android-app` — код мобильного приложения
- `/website` — код веб-сайта
- `/scripts` — вспомогательные скрипты

## Проекты внутри экосистемы
- **MOZG Universe** — основной бренд свечей и декора
- **Адвокат в бардачке** — юридическая консультация для водителей (в разработке)

## Статус
MVP, активный поиск поставщиков и разработка визуального стиля.
```

1. Commit changes.

4.4. SECURITY.md (политика безопасности)

1. Add file → Create new file
2. Имя: SECURITY.md
3. Вставьте:

```markdown
# Политика безопасности MOZG Universe

## Конфиденциальная информация
Никогда не коммитьте:
- Файлы `.env`, `.env.*`
- Ключи API, пароли, сертификаты
- Личные данные пользователей

## Сообщение об уязвимостях
Если вы нашли уязвимость — напишите на security@mozg-universe.com (в будущем).
```
